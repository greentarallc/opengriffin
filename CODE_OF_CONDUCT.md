# Code of Conduct

OpenGriffin follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

## TL;DR

Be civil. Assume good faith. Help newcomers. Disagree on the work, not the person. If someone tells you their experience of an interaction was negative, take it seriously regardless of intent.

## Scope

These standards apply in all project spaces — GitHub issues, pull requests, discussions, the project Telegram channel, and any external community space when you're representing OpenGriffin.

## Enforcement

Report concerns to **conduct@opengriffin.com**. Reports are confidential. Maintainers will respond within a week with what they intend to do.

Possible outcomes range from a private clarification, to a public correction, to a temporary or permanent ban from project spaces — proportional to severity and pattern.

## Maintainer accountability

Maintainers are held to the same standards as contributors, plus the obligation to apply this Code consistently and without favoritism. If you believe a maintainer is in breach, escalate to a different maintainer, or to **conduct@opengriffin.com** if that fails.
